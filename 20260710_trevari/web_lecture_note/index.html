내 일에 바로 써먹는 AI 자동화

ChatGPT도, Claude도 써봤습니다. 그런데 막상 내 일에 가져와 붙이려 하면 어디서부터 손대야 할지 막막했을 거예요. 매일 반복되는 자잘한 일은 여전히 손으로 하나하나 처리하고 계실 거고요. 이 모임은 AI로 더 많은 일을 해내기가 아니라, 어떤 일은 자동화하고 어떤 일은 사람의 손에 남길지 함께 선을 긋는 모임입니다.

발언이 끝날 때마다 요약해서 붙여두면, 누가 어떤 이야기를 했는지 흐름을 놓치지 않습니다. 🎤 버튼으로 실시간 받아쓰기를 켜거나 직접 입력하세요. (AI 요약에는 아래 요약 리포트 박스의 OpenAI API 키를 사용하며, 키가 없으면 원문 그대로 붙습니다)

🌟 모임 전체 내용 요약 리포트 (OpenAI API)

'전체 요약하기' 버튼을 누르면 오늘 각 멤버들이 작성한 워크플로우 설계 아이디어를 모두 모아, OpenAI GPT-4o-mini 모델이 아주 깔끔한 회의록으로 정리해 줍니다.

※ 우측 상단에 OpenAI API 키를 입력해 주세요. (입력된 키는 브라우저에 안전하게 자동 저장됩니다)

시즌 목표

시즌이 끝날 즈음, 우리 손에 남는 것들입니다.

✅ 하나의 실전 자동화
적어도 내 일을 괴롭히던 반복 업무 하나가 실제로 돌아가는 자동화로 바뀝니다.
✅ 나만의 워크플로우
'나만의 워크플로우 설계안' 한 장을 완성하여 앞으로의 든든한 기준점으로 삼습니다.

4개월의 여정

무엇을 다루고 어떻게 발전시킬 것인지, 한 달에 한 번씩 함께 고민하고 만들어 갑니다.

1회차

자동화의 공통 언어 잡기

내 일을 '트리거 → 처리 → 출력 → 검토'로 쪼개보고, 자동화에 올릴 후보 업무를 정합니다.

  • 01 · 공통 언어 잡기 (7/10)
2회차

'바쁘기만 한 일' 걷어내기

메일 분류, 단순 입력처럼 바쁘기만 한 가짜 노동을 찾고 자동화로 넘길 대상을 확정합니다.

  • 02 · 일 걷어내기 (8/14)
3회차

첫 자동화 v1 만들기

무엇을 남기고 넘길지 기준을 정한 후, 작은 버전(v1)의 설계도를 그려보고 첫 연동을 해봅니다.

  • 03 · 첫 자동화 v1 (9/11)
4회차

아낀 시간을 어디에 쓸 것인가

자동화로 되찾은 시간을 어떻게 쓸지 논의하며, 최종 완성된 워크플로우를 서로 시연하고 공유합니다.

  • 04 · 시간의 쓰임 (10/9)
