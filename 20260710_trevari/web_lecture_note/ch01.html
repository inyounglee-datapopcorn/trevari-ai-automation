n8n이 다 해줌

저자: 이인영

자동화를 이야기하려면 먼저 우리에게 공통 언어가 필요해요. 첫 모임에서 함께 읽을 『n8n이 다 해줌』이 그 출발점입니다. 비개발자도 n8n으로 데이터 수집, API 연동, AI Agent 워크플로우 자동화의 기본 구조를 이해할 수 있도록 쓴 입문서예요. 기능을 외우는 책이 아니라, 반복되는 업무를 작은 단계로 나누고 다시 연결하는 사고방식을 익히는 책입니다.

🤖 Claude Code & n8n MCP 연동 (심화 실습)

n8n 워크플로우가 수십 개로 늘어나면 관리하기 힘들어집니다. 터미널에서 자연어(말) 한마디로 n8n을 제어하고 자동화할 수 있도록 Claude Code와 n8n MCP를 연결해 봅니다.

1

Claude Code 설치: 터미널을 열고 npm install -g @anthropic-ai/claude-code 명령어로 설치 후 claude를 실행하여 로그인합니다.

2

n8n API 키 발급: n8n 워크스페이스 좌측 하단의 Settings > n8n API 메뉴에서 새로운 API 키를 생성하고 안전하게 복사합니다.

3

MCP 서버 연결: 터미널에 claude mcp add n8n -- npx -y n8n-mcp-server를 입력하고, 안내에 따라 n8n 서버 URL(예: https://...)과 복사한 API 키를 입력합니다.

4

첫 자연어 명령 성공하기: 연동이 끝났다면, "현재 n8n에 있는 워크플로우 목록 보여줘" 라고 명령하여 성공적으로 n8n 데이터를 읽어오는지 확인합니다.

💡 MCP 연동이 왜 중요한가요?
단순히 남이 만든 워크플로우를 복붙하는 것을 넘어, AI가 내 n8n 환경을 이해하고 스스로 워크플로우를 생성, 수정, 에러 분석까지 해주는 '자동화 관리자'를 고용하는 것과 같습니다.
